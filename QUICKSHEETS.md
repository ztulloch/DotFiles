# tmux
```
tmux a -t # - attach to tmux session
```
**_CTRL b n_** next pane
**_CTRL b p_** previous pane
**_CTRL b (_** switch to next tmux session
**_CTRL b )_** switch to previous tmux session
**_CTRL b d_** detach session
